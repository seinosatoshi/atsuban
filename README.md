あつまれ！インディーズバンド
## サイト概要
ライブハウスでの活動が制限されているバンドマン達が集う場所。こんな世の中だからこそ全国にいるライブバンドがオンラインで表現できる場所を作りました。自身のバンド情報を登録し、音源・動画をアップロードしてユーザーからのエールをGETしよう！

### サイトテーマ
インディーズバンドよ、エールを抱け！

### テーマを選んだ理由
バンドをしていたときにこんなサイトがあったらいいなと思ったことがきっかけ。

### ターゲットユーザ
１０〜２０代のバンドマンがメイン

### 主な利用シーン
自身のHPを持っていない、他のバンドマンの情報を知りたい方たちがオンラインで情報共有できる。
